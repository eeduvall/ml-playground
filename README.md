# 🚀 Machine Learning Lab: Experimental Sandbox
A curated collection of machine learning experiments, model evaluations, and data explorations. This repository serves as a personal portfolio for testing state-of-the-art algorithms against real-world datasets.

## 📋 Overview
The goal of this repository is to document my journey through various ML architectures—ranging from classic statistical models to deep learning. This repository is structured by **algorithm type** to showcase technical depth across different model classes.


## 🛠 Tech Stack
* **Languages:** Python
* **Libraries:** Scikit-Learn, Pandas, NumPy, Matplotlib, XGBoost
* **Tools:** Jupyter Notebooks, Docker (for environment reproducibility)
