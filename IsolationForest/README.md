# LEAD Energy Dataset Analysis

This code is used against the **LEAD energy dataset** found at:  
[Kaggle: Energy Anomaly Detection](https://www.kaggle.com/competitions/energy-anomaly-detection/data?select=test_features.csv)

### Model Results
* **Isolation Forest Detection:** 4.62% anomalies
* **Contest Winner Benchmark:** ~5% anomalies

> **Conclusion:** The model works quite well out of the box, as the results are highly consistent with the winning entry.
